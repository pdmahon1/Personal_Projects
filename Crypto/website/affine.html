What is an affine cipher?

The "simple" answer to this question is that every letter, when converted to an equivalent number, is multiplied by a constant number and then translated (or "shifted") by another constant number. In an affine cipher, the equation is commonly written as n ↦ αn + β ≡ K (mod 26), which resembled the point-slope equation of a line.

Fun fact: setting α = 1 makes this cipher a Caesar cipher!
